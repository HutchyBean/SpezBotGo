# SpezBot
### A discord bot that plays music for me and copains

## Features
- Plays Music
- Uses slash commands
- Has a fair queue system
- Better written than C# version

## Commands
- /play
- /playnext
- /playnow
- /skip
- /queue
- /echo
- /pause
- /resume
- /seek
- /shuffle
